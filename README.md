# izukampusbot
a telegram bot for izu kampus website

## Requirements

- Node 6+
- npm or yarn
- mysql
- telegram bot token

## Installation

1. `yarn install`
2. `cp credentials.example.js credentials.js` and fill it
3. `create new mysql database`
4. `node index.js`
